## UWOW Develop Helper

Программа написана для разработчиков проэкта UWOW, что бы автоматизировать некоторые рутинные процессы в разработке проэкта.

**Возможности:**

* [Генерация бозового класса скриптов способностей персонажей.](Docs/CodeGenerator/CodeGenerator.md)
* [Обновление базы данных в один клик.](Docs/DBUpdater/DB_Updater.md)
* [Контроль зарплаты и создание ежемесячных отчетов в один клик о проделанной работе. (Тест)](Docs/Salary/Salary.md)
