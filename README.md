# UWOW Develop Helper

Программа написана для разработчиков проэкта UWOW, что бы автоматизировать некоторые рутинные процессы в разработке проэкта.

**Возможности:**

* [Генерация бозового класса скриптов способностей персонажей.]()
* [Обновление базы данных в один клик.]()
* [Контроль зарплаты и создание ежемесячных отчетов в один клик о проделанной работе. (Тест)]()
